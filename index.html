Organic Shape Animations

made with AnimeJS and JQuery

View on GitLab Download

Original idea by @Tympanus revisited by @kzl____

How to get them ?

  1. Include JQuery and AnimeJS

    If you haven't done that already, include the latest JQuery and AnimeJS in your project.

  2. Include the CSS

    To enable the socials buttons, just include the stylesheet-socials.css file into your project.

  3. Add some new shapes!

    Start using BlobJS library and go to js/blob.js. You will find some BlobShape() function call. These are meant to add shapes in the list of available shapes including an shape ID. You will then be able to use it when calling a .blob class. There blobs can be parametrized including : 

    data-href, data-shape, data-image, ..
